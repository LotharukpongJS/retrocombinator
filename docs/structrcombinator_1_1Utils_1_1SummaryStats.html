template<typename T, typename T_cont>
struct rcombinator::Utils::SummaryStats< T, T_cont >

Plain-Old-Data struct to store summary statistics for a list of values.

Is templated on T and T_cont, where T is a numeric type, and T_cont is a numeric type that can store fractional parts. Stores the quartiles and the mean for a list of values.
