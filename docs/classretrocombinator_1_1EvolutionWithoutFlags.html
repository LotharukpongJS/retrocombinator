Detailed Description

A simulation where all sequences are active.

In this class, a sequence beocoming inactive is equivalent to a sequence dying and being removed from the simulation and sequence pool.

◆ copy_numbers

tree_type retrocombinator::EvolutionWithoutFlags::copy_numbers
private

The phyolgenetic tree of transposons for all times.

This assumes there is only one family for the time being. copy_numbers[t][i] = (a, b) means that at time t, the ith sequence is got by recombining sequence a and b at time t-1. If b = SAME_SEQ it was just that sequence a. This is different to the case when a=b (where a new sequence, with a new sequence tag, was created but it is identical to a). If recomb_mean is 0, we pick sequence a to burst.
