Breaks a set into two clusters.

Parameters
nthe number of data points
distthe nXn distance matrix between the data points Takes a distance matrix D, where D(i, j) is the distance between i and j, with D(i, i) being 0 for all i. Returns a vector of indices that belong to the first cluster (the second cluster corresponds to everything else).

Algorithm: Start with n clusters Compute the cluster distance between all pairs of clusters Where cluster distance is: average distance of points from one cluster to the other Pick the two closest clusters, and merge them
